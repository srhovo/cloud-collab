# 阶段3A安全边界

1. 只读接口只接受 GET、HEAD、OPTIONS。
2. 客户端请求使用 `credentials: omit`，不携带 Cookie 或 Authorization。
3. 页面启动通过 `setTimeout` 异步检查，不等待服务器。
4. health和protocol失败只改变瞬态服务器状态，不进入本地数据加载失败链路。
5. public-version结果只保存在内存中的 `lastPublicVersion`，不写入价格库、老板记忆、同步Store或待上传队列。
6. 页面启动不会创建deviceId，也不会生成六个云协作localStorage key。
7. 测试目录只登记一个空公共库身份，版本为0，不包含任何价格、名字、老板或用户记录。
8. 标准备份、导入和旧数据迁移逻辑与8.2.26保持逐类代码一致。
9. 本阶段没有submission、管理员写接口、KV写入或Blob写入。
10. HTML、仓库和接口代码中没有硬编码Token、密码或密钥。
