# 阶段7W：正式管理员设备治理

## 范围

把阶段5D已验证的设备治理核心接到正式管理员会话与生产公共Blob。提供设备列表、详情、可信状态维护和设备状态维护；不改变公共价格、规则、候选版本或稳定版本。

## 独立开关

```text
CLOUD_PRODUCTION_DEVICE_GOVERNANCE_ENABLED=0
```

缺失时按0处理；只能在生产总开关和正式管理员身份开启后使用；独立于普通与敏感审核开关；需要独立的`CLOUD_ADMIN_DEVICE_REF_SALT`。生产模板默认保持0。

## 路由

```text
GET  /api/admin/devices
GET  /api/admin/devices/detail?id=<device-ref>
POST /api/admin/devices/trust
POST /api/admin/devices/revoke-trust
POST /api/admin/devices/block
POST /api/admin/devices/unblock
```

路由按`CLOUD_PRODUCTION_ENABLED`选择正式或隔离预览处理器。生产模式中子开关关闭时返回503，不回退预览。

## 身份和存储

正式处理器只接受`cloud-collab-admin-production`会话，固定使用`cloud-collab-production-v1`和管理员HTTPS来源。GET要求同站；POST还要求精确同源。错误来源在正文解析和Store创建前阻断。

## 脱敏

响应只使用不可逆`deviceRef`，不返回设备ID、令牌、令牌摘要、Blob Key、管理员Cookie或真实秘密。

## 核心语义

复用阶段5D不可变事件、请求索引和审计：

- 可信状态变更保留历史；
- 暂停设备会同时清除可信状态，并立即影响设备认证；
- 恢复设备不会自动恢复可信状态；
- 每个请求有不可变请求ID和请求Hash；
- 精确重放返回原结果，不同正文复用请求ID会冲突。

能力边界：

```text
publicMutationAllowed=false
stablePromotionAuthorized=false
syntheticFixtureOnly=false
```

专项CI覆盖独立开关、生产会话、生产Store、脱敏投影、同源阻断、生产模式不回退预览，以及阶段5D完整治理和设备认证回归。完整Node和浏览器矩阵继续运行。
