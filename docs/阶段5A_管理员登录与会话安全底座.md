# 阶段5A：管理员登录与会话安全底座

## 基线与范围

- 基线：`main@3f6910f5503df3268fd202e4143570ef61089dee`。
- 分支：`agent/stage5a-admin-auth-foundation`。
- 管理员页面与普通码单器页面完全分离；普通用户端不链接、不加载管理员页面。
- 本阶段只建立登录、登出、会话校验和登录限流。
- 审核队列、批准、拒绝、修改后批准、可信设备、封禁、回滚、导出及公共版本修改全部保持关闭。
- 阶段4F分支、页面、清理路由、密钥和Blob对象均不复用。

## 默认关闭与环境变量

`CLOUD_ADMIN_PREVIEW_ENABLED` 在仓库中必须默认为 `0`。只有独立管理员预览部署可以临时设为 `1`。

`CLOUD_ADMIN_PUBLIC_ORIGIN` 必须显式设置为管理员预览页面的HTTPS来源；只允许协议和域名，不得包含路径、查询参数、预览token或用户信息。EdgeOne同一项目的预览域名可以仅轮换末尾12位部署标识；项目前缀与`.edgeone.cool`后缀必须完全一致。其他平台仍要求Host与固定公开来源完全一致。

管理员用户名、密码、会话密钥和限流盐值只允许进入EdgeOne私密环境变量。密码至少16字节；会话密钥与限流盐值各至少32字节，三者必须互不相同。任何真实值都不得进入GitHub、HTML、网址、日志、截图或聊天。

管理员预览登录还会拒绝以下任一开关为 `1` 的环境：

- `CLOUD_WRITE_PREVIEW_ENABLED`
- `CLOUD_AUTO_APPROVAL_PREVIEW_ENABLED`

登录限流只使用硬锁命名空间 `cloud-collab-admin-preview-v1`，不得与公共数据或阶段4预览Blob共用。

## API

```text
POST /api/admin/auth/login
GET  /api/admin/auth/session
POST /api/admin/auth/logout
```

- 登录与登出只接受同源HTTPS请求，不开放CORS。
- 登录体严格限制为 `schemaVersion / username / password`，最大2KB。
- 每个“用户名 + 客户端地址”在10秒不可变槽内最多尝试一次；Blob Key只含加盐Hash，不保存原始用户名或IP。
- 登录成功后服务器签发15分钟HS256会话，只通过 `HttpOnly; Secure; SameSite=Strict; Path=/api/admin` Cookie传递。
- JSON响应不返回Token、密码、会话密钥或限流盐值。
- 会话失效或篡改时服务器同时清除Cookie。

## 管理员页面

临时预览路径：

```text
/admin-preview.html
```

页面不使用LocalStorage、SessionStorage或IndexedDB；密码输入在请求发出前立即从输入框清空。阶段5B开始前，该页面只能显示身份状态，不能读取审核队列。
