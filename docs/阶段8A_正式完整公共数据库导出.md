# 阶段8A：正式完整公共数据库导出

## 目标

阶段5F预览导出只包含普通精确价格。阶段8A新增生产专用导出，不修改阶段5F兼容链，并完整包含：

```text
exact_price
playable_name
boss_profile
rank_range_rule
surcharge_rule
gift_rule
墓碑
普通公共事件历史
敏感公共事件历史
```

ZIP中的`snapshot/all.json`是恢复公共数据库的权威快照，事件文件用于审计。

## 独立门禁

```text
CLOUD_PRODUCTION_EXPORT_ENABLED=1
```

依赖生产、只读同步和管理员身份，不依赖普通/敏感写入、人工审核或设备治理。关闭导出时生产项目返回503，不回退阶段5F预览。

## 路由

```text
GET  /api/admin/exports/summary
POST /api/admin/exports/download
```

正式导出使用公共生产Store、`group_see/lib_see_cz`、`cloud-collab-admin-production`会话和`CLOUD_ADMIN_EXPORT_AUDIT_SALT`。

## 请求与幂等

下载请求：

```json
{
  "schemaVersion": 1,
  "requestId": "exrq_v1_...",
  "confirmation": "EXPORT_FULL_PUBLIC_DATABASE"
}
```

同一请求ID在公共版本未变化时返回同一包；公共版本变化后返回409并要求新请求ID，避免同一请求悄悄对应不同备份。请求索引、导出决策和审计记录均不可变。

## 隐私边界

导出包含公共数据库与脱敏公共事件历史，不包含管理员密码、会话、设备令牌、登录限流、访问密钥或盐值。响应声明`privateCredentialsIncluded=false`、`realSecretValuesExposed=false`和`stablePromotionAuthorized=false`。
