- 本阶段分支：`agent/stage5d-admin-device-governance`，必须始终从上述新main独立推进。

## 本阶段唯一主目标

建立管理员设备治理能力，范围只包括：

1. 管理员只读设备列表与单设备详情投影；
2. 将设备设为可信、撤销可信；
3. 封禁设备、解除封禁；
4. 每个动作的幂等请求、不可变治理事件和不可变审计记录；
5. 设备认证与自动审核读取同一份有效治理状态，避免“后台显示已封禁但提交仍可用”或“已撤销可信仍单设备自动批准”。

本阶段不包含：

- 公共数据回滚；
- 数据导出或迁移包；
- 区间、加价、礼物、删除、直属变化等敏感规则提交；
- 正式用户价格库写入；
- 正式环境管理员能力启用；
- 修改冻结的码单器8.2.25或普通用户页面。

## 默认关闭与作用域

新增独立门禁，仓库模板必须默认为：

```text
CLOUD_ADMIN_DEVICE_GOVERNANCE_PREVIEW_ENABLED=0
```

只有同时满足以下条件才允许读取或写入：

- 阶段5A管理员会话有效；
- 当前浏览器请求通过既有同项目HTTPS与同源校验；
- 管理员预览、审核读取及设备治理门禁按预期组合开启；
- 公共写入和正式自动批准门禁保持关闭；
- Blob Store严格锁定`cloud-collab-preview-v1`；
- 只处理合成预览设备，不允许访问正式用户或其他命名空间。

阶段5D不得顺带开启阶段5C之外的公共变更能力。可信设备只允许沿用既有规则：可信设备可对“新增普通精确价格”形成单设备自动批准资格；不得单独批准已有价格修改。

## 隐私投影

管理员响应不得返回：

- 明文设备令牌；
- tokenHash或令牌索引Key；
- Blob Key和存储路径；
- 完整submissionId、idempotencyKey或requestHash；
- 历史记录、订单、聊天、备注、自定义比例、布局或使用习惯；
- 管理员密码、会话密钥、限流盐值或Cookie内容。

设备操作必须使用服务端生成的不可逆`deviceRef`，不能把Blob路径或tokenHash当作操作ID。界面只显示必要的脱敏设备标签、昵称标签、注册/过期时间、最后客户端版本、可信状态和封禁状态。

## 状态与动作

允许动作固定为：

```text
trust
distrust
block
unblock
```

规则：

- `trust`对已可信设备为幂等恢复，不新增第二份有效状态；
- `distrust`立即使后续自动审核不再把该设备视为可信；
- `block`立即使后续设备认证和候选提交失败关闭；
- `unblock`只解除封禁，不自动恢复可信；
- 封禁不得删除历史候选、公共事件或审计记录；
- 解除封禁不得补发新的设备令牌；
- 同一请求重放必须返回同一治理事件与审计ID；
- 并发相反动作必须形成单一、可验证的有效顺序，不能出现两个并存的当前状态。

## 与既有代码的兼容审计

正式实现前必须先冻结以下代码事实：

1. `src/server/device_registration_v1.js`当前只验证令牌索引和设备档案，尚未读取封禁状态；
2. `src/server/auto_approval_engine_v1.js`当前通过`devices/trusted/<deviceId>.json`读取可信状态；
3. 阶段5D必须让认证和自动审核读取统一治理状态，不能维护两套会漂移的判断；
4. 既有阶段4D、5A、5B、5C测试必须原样继续通过；
5. 新治理记录格式必须支持强一致读取、幂等恢复、并发冲突检测和后续一次性验收清理白名单扩展。

在完成上述兼容审计前，不直接改写现有可信设备记录，也不通过删除令牌索引来模拟可解除的封禁。

## 预计接口与页面

接口名称在实现审查时最终冻结，候选范围为：

```text
GET  /api/admin/devices
GET  /api/admin/devices/detail?id=<opaque-device-ref>
POST /api/admin/devices/trust
POST /api/admin/devices/distrust
POST /api/admin/devices/block
POST /api/admin/devices/unblock
```

隔离预览页面候选：

```text
/admin-device-governance-preview.html
```

页面不得使用LocalStorage或SessionStorage保存管理员凭据、设备操作请求或任何密钥。

## 最低验证门槛

- 设备投影隐私测试；
- 信任、撤销信任、封禁、解封的单元测试；
