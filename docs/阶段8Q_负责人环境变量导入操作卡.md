```dotenv
CLOUD_ADMIN_PASSWORD=<本机生成>

CLOUD_PRODUCTION_RATE_LIMIT_SALT=<本机生成>

CLOUD_ADMIN_ROLLBACK_REF_SALT=<本机生成>

```

使用仓库中的：

```text
tools/production-secret-generator.html
```

生成规则：每项至少32字节、八项完全不同；仅保存到密码管理器和EdgeOne环境变量，不发送到聊天、不截图、不提交GitHub。

## EdgeOne导入位置

```text
EdgeOne Makers
→ cloud-collab
→ 项目设置
→ 环境变量
→ 生产环境
```

若控制台支持批量导入，使用生成器导出的完整文本；否则逐项新建。涉及密码、Key、Secret、Salt的八项设为加密/私密变量。其余固定值可以使用普通变量。

保存变量后平台可能自动触发或要求重新部署。部署完成后只验证页面仍能访问以及所有生产能力保持关闭；此阶段不执行Blob初始化、不把任何开关改为`1`。
