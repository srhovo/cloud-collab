# 阶段8A：正式迁移导出

## 目标

在正式只读、普通/敏感审核、设备治理和公共数据回滚代码已经闭环后，提供一个不依赖EdgeOne专有结构的完整迁移ZIP。该包用于备份、平台迁移和灾难恢复准备，不修改公共数据。

## 独立生产门禁

```text
CLOUD_PRODUCTION_EXPORT_ENABLED=0
```

该开关默认关闭。启用条件为：

```text
CLOUD_PRODUCTION_ENABLED=1
CLOUD_PRODUCTION_READ_SYNC_ENABLED=1
CLOUD_ADMIN_PRODUCTION_ENABLED=1
CLOUD_PRODUCTION_EXPORT_ENABLED=1
```

迁移导出不要求开启普通提交、自动审核、敏感提交、人工审核或设备治理。它只依赖只读公共快照和正式管理员身份。

生产总开关为1时，即使导出子开关关闭，也只返回503，不回退阶段5F合成预览导出。

## API

```text
GET  /api/admin/exports/summary
POST /api/admin/exports/download
```

路径与阶段5F保持兼容，由`admin_export_mode_dispatch_v1.js`按生产总开关选择正式或预览处理器。

正式下载确认词：

```text
EXPORT_PRODUCTION_MIGRATION_PACKAGE
```

## 数据覆盖

生产包直接读取统一公共快照，覆盖：

```text
exact_price
playable_name
boss_profile
rank_range_rule
surcharge_rule
gift_rule
显式删除墓碑
```

`libraries/lib_see_cz.json`保留完整统一快照，包括：

- `group_see / lib_see_cz`；
- 普通基线版本；
- 当前公共版本、游标和快照版本；
- 全部当前记录；
- 全部当前墓碑。

其他目录提供按业务类型拆分的可迁移视图。

## 完整性

- 每个文件在manifest中记录字节数和SHA-256；
- packageId由作用域、公共版本及全部文件描述符稳定计算；
- ZIP文件名使用UTF-8标志；
- 统一快照包含重复业务键、记录/墓碑冲突或无效结构时失败关闭；
- 公共版本发生变化时，旧requestId不静默导出新内容。

## 幂等与审计

下载请求包含：

```json
{
  "schemaVersion": 1,
  "requestId": "exrq_v1_...",
  "confirmation": "EXPORT_PRODUCTION_MIGRATION_PACKAGE"
}
```

服务器把`requestId`和管理员身份转为不可逆请求令牌与摘要：

- 相同请求跨时间精确重放，返回同一packageId和相同ZIP字节；
- 请求索引、决定和审计均不可变；
- 审计只保存不可逆actorTag，不保存用户名、密码或会话；
- 公共版本已经变化时返回`PRODUCTION_EXPORT_BASE_MOVED`，必须换新requestId；
- 不在Blob中长期保存ZIP字节，避免重复大对象和敏感备份滞留。

## 安全边界

- 正式接口只接受`cloud-collab-admin-production` issuer会话；
- POST必须精确同源；
- 跨站请求在正文解析和公共Blob创建前阻断；
- 响应和ZIP不包含设备ID、设备令牌、令牌Hash、提交ID、幂等键、内部Blob路径或真实密钥；
- 只访问`cloud-collab-production-v1`；
- 不写普通或敏感公共事件，不执行回滚，不改变公共版本；
- 所有响应继续标记`stablePromotionAuthorized=false`。

## 与阶段5F的关系

阶段5F合成预览导出保持原样：

- 继续硬锁`group_fixture / lib_receive_fixture`；
- 继续使用阶段5A预览会话；
- 原测试和浏览器回归不变。

阶段8A是独立生产实现，不把正式作用域放宽到阶段5F预览核心。
